Import as Logical Model
Strip text
  • Paste the XML or Json into the box and click the Import button (top right).
  • Json can be a single resource, a bundle of resources or a Logical Model. A Bundle must have 'fullUrl' set on each resource
  • For Logical Models, make sure to check the 'Logical Model' checkbox, or it will be imported as a StructureDefinition
  • XML must be a bundle, with a type of 'collection', no Bundle Id, and the conformance server must support the Bundle endpoint for both Json and XML
