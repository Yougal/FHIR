This section of the Guide is the documentation.

The Table of Contents (TOC) is displayed as a tree on the left side of the screen. Selecting an item will display the contents in this pane. Documentation contents must be HTML - for example a Google document that is maintained separately to the guide. (Note that the TOC also includes the FHIR artifacts that are a part of this Guide)

To add a new documentation node, follow these steps:

  1. Determine what the Url to the page is (it must be accessible from the Internet)
  2. Select the 'parent' node in the tree. The page you insert will become a 'child' of that node. For a 'top level' page select the 'Documentation' node.
  3. Click the plus ( ) symbol at the top right of the tree. A dialog box will appear.
  4. Enter the Url to the page you are adding.
  5. In the dialog, enter the page title (which will appear in the Table of Contents) and the Url of the page. Click save to save.
  6. The TOC will be updated.

The other options for editing the TOC are:

  • - shows the dialog where the title and url can be changed
  • Remove the page from the TOC. Note that this doesn't delete the page, just the reference in the TOC.
  • Moves the selected item down the TOC (Unless it is already at the bottom).
  • Moves the selected item up the TOC (Unless it is already at the top).
  • Inserts a new page into the TOC as a child of the currently selected item.
