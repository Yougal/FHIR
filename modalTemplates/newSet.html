Create new scenario

It is possible to associate a scenario with a patient that already exists on the Data Server. This allows you to select existing Resource instances to add to the scenarios, update those instances and also create new ones. You can also add a new patient with associated sample data from the same selection dialog.

It is an optional step, and can be done later (there's an add patient option on the Description tab)
